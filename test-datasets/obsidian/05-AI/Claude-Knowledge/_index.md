# Claude Knowledge Base

Knowledge generated by Claude Code sessions, automatically synced from AIProjects.

**Source**: `AIProjects/.claude/agent-output/`, `knowledge/notes/`, `.claude/context/patterns/`
**Sync**: Runs at session end via `sync-knowledge-to-obsidian.sh`
**Direction**: One-way (AIProjects â†’ Obsidian). Originals stay in AIProjects.

## Folders

| Folder | Source | Content |
|--------|--------|---------|
| `research/` | `agent-output/results/deep-research/` | Deep research reports with multi-source analysis |
| `session-notes/` | `knowledge/notes/` | Session discoveries, research notes, methodology docs |
| `patterns/` | `.claude/context/patterns/` | Architectural patterns and workflows (full archive) |
| `discoveries/` | `agent-output/results/` (non-research) | Agent findings: troubleshooting, code analysis, etc. |
