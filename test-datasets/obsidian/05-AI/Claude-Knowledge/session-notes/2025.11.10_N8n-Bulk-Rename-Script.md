## How to Execute

Each workflow update uses the `n8n_update_partial_workflow` MCP tool with operations:

```javascript
n8n_update_partial_workflow({
  id: "workflow-id",
  operations: [
    { type: "updateName", name: "New Name" },
    { type: "removeTag", tagId: "old-tag-id" },
    { type: "addTag", tagName: "New Tag" }
  ]
})
```
