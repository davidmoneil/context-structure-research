## Technical Specifications

### Atlas A1 - Mining Configuration
- **Height**: 4.2 meters
- **Weight**: 8,400 kg
- **Payload**: 12,000 kg
- **Power**: Hydrogen fuel cell (72-hour endurance)
- **Locomotion**: 8-wheel articulated drive
- **Manipulation**: Dual 6-DOF arms (2,000 kg lift each)

### Atlas A2 - Construction Configuration
- **Height**: 3.8 meters
- **Weight**: 6,200 kg
- **Payload**: 8,000 kg
- **Power**: Battery electric (24-hour endurance)
- **Locomotion**: Tracked + bipedal hybrid
- **Manipulation**: Tool-change system (47 attachments)

### Atlas A3 - Disaster Response
- **Height**: 2.4 meters
- **Weight**: 1,800 kg
- **Payload**: 400 kg
- **Power**: Battery + solar (extended endurance)
- **Locomotion**: Quadruped adaptive
- **Special**: Radiation/chemical hardened

### AI Capabilities
- **Core**: SCE Platform v3.2
- **Perception**: LiDAR, stereo vision, thermal, radar
- **Navigation**: GPS-denied capable, SLAM
- **Decision**: Real-time task planning, hazard avoidance
- **Communication**: Mesh network, satellite backup
